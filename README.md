## üì± Core Pages

| Page             | Features                                                       |
|------------------|----------------------------------------------------------------|
| **Landing**       | Mission, Sign Up / Log In                                     |
| **Auth**          | GitHub + Firebase/Auth0 login and sign up                     |
| **Profile Setup** | Add bio, skill tags, GitHub/LinkedIn links                    |
